# DAPPER PROJECT

ℹ️ Dapper Orm aracını kullanarak trafikteki araçların bilgilerini listeleme,çeşitli parametreler ile filtreleme ve grafik üzerinde görüntüleme işlemleri gerçekleştirilmiştir.Veri seti olarak Kaggle.com sitesi üzerinden 1946 - 2020 yılları arasında trafikte yer alan 4 milyondan fazla araç ile oluşturulmuş veri büyük veri setini kullanmaktadır.
